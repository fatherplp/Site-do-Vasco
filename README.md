# Site do Vasco  

Este repositório contém o código-fonte para o site dedicado ao Vasco da Gama.  
O objetivo é criar uma plataforma informativa com foco nas contratações, visão geral do clube e um design que destaca as cores branco e vermelho, com a Cruz de Malta como elemento de destaque no background.  

## Tecnologias Utilizadas  

- **HTML**: Para estruturação do conteúdo do site.  
- **CSS com TailwindCSS**: Para estilização eficiente e moderna.  
- **JavaScript**: Para funcionalidades interativas e dinâmicas.  

## Status do Projeto  

O projeto está em desenvolvimento contínuo e novas funcionalidades e melhorias estão sendo implementadas regularmente.  

## Como Executar o Projeto  

Siga os passos abaixo para visualizar o projeto em sua máquina local:  

1. **Clone o Repositório**  
   ```bash
   git clone https://github.com/joaodorast/Site-do-Vasco.git
   ```  

2. **Acesse o Diretório do Projeto**  
   ```bash
   cd Site-do-Vasco
   ```  

3. **Abra o Arquivo HTML no Navegador**  
   Abra o arquivo `index.html` diretamente no navegador para visualizar o site.  

4. **Servidor Local (Opcional)**  
   Para uma experiência mais fluida, utilize um servidor local:  
   - Instale a extensão **Live Server** no Visual Studio Code.  
   - Clique com o botão direito no arquivo `index.html` e selecione **Open with Live Server**.  

## Estrutura do Projeto  

- **`index.html`**: Página principal do site.  
- **`/css`**: Arquivos de estilização e configuração do TailwindCSS.  
- **`/js`**: Scripts JavaScript para as funcionalidades do site.  
- **`/assets`**: Recursos estáticos como imagens, ícones, entre outros.  

## Funcionalidades  

- **Informações sobre o Vasco da Gama**: Dados históricos e atuais sobre o clube.  
- **Atualizações sobre Contratações**: Informações de novos jogadores e movimentações do clube.  
- **Background com a Cruz de Malta**: Design inspirado nas cores tradicionais do clube (branco e vermelho).
